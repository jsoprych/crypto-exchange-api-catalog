Initial commit: Vendor API Specification Generator

A robust, extensible system for discovering and cataloging cryptocurrency 
exchange APIs. Generates comprehensive, machine-readable JSON specifications 
for data ingestion applications.

=== CORE FEATURES ===

• SQLite-Backed Architecture
  - 11 relational tables with full audit trail
  - 4 pre-built views for common queries
  - Complete change detection and versioning
  - Queryable via SQL for powerful analysis

• Multi-Vendor Support
  - Extensible adapter pattern for any exchange
  - Coinbase Exchange fully implemented (776 products)
  - Ready for Binance, Kraken, and others

• Comprehensive API Discovery
  - REST endpoints with parameters and schemas
  - WebSocket channels with message formats
  - Trading products/symbols with metadata
  - Product-to-feed relationship mapping

• Flexible JSON Export
  - Python format (snake_case) for Python/REST
  - Go format (camelCase) for Go/JavaScript
  - Configurable output options
  - Complete specifications with vendor metadata

• Command-Line Interface
  - init: Initialize database schema
  - discover: Run API discovery for a vendor
  - export: Generate JSON specifications
  - list-vendors: Show configured vendors
  - query: Execute SQL queries directly

=== IMPLEMENTATION DETAILS ===

Architecture:
  Vendor Adapters → SQLite Database → JSON Export
  
  Three-layer separation ensures:
  - Vendor-specific logic isolated in adapters
  - SQLite as single source of truth
  - Multiple export formats from one source

Code Quality:
  • All functions <50 lines (modular design)
  • Complete docstrings on every function
  • PEP8 compliant Python code
  • Comprehensive error handling
  • Structured logging (console + file)

Testing:
  ✓ Coinbase discovery: 776 products, 10 REST endpoints, 5 WS channels
  ✓ JSON export (both formats) verified
  ✓ SQL queries functional
  ✓ All CLI commands tested
  ✓ Duration: ~20 seconds for full discovery

=== USAGE ===

# Initialize database
python3 main.py init

# Discover Coinbase API
python3 main.py discover --vendor coinbase

# Export specification (Python format)
python3 main.py export --vendor coinbase --format snake_case

# Export specification (Go format)
python3 main.py export --vendor coinbase --format camelCase

# Query database
python3 main.py query "SELECT * FROM products WHERE base_currency = 'BTC'"

# Use SQL query library
sqlite3 data/specifications.db < sql/queries/01_vendor_analysis.sql
